## Packages
react-tsparticles | For the futuristic neural network / cloud node background visualization
@tsparticles/slim | Slim engine for the particle animations
framer-motion | Essential for the floating nodes, scroll animations, and premium transitions
react-hook-form | For the contact form
@hookform/resolvers | For Zod validation in the contact form
lucide-react | For cloud and infrastructure icons

## Notes
Static assets can be generated purely via CSS/SVG/Icons.
Contact form requires POST /api/messages endpoint to be operational.
Navigation relies on anchor tags (#id) since it's a single page portfolio.
